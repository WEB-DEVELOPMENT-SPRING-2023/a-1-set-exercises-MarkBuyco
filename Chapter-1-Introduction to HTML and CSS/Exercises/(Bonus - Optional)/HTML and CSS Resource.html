Input

Output

This is heading 1.

<b>

<i>
<i> for italics, which will italize the text in html.
Input
<p> This text is in <i> italics. </i> </p>
Output

This text is in italics. 



<ul>
<ul> stands for unordered list and is used to make a bulleted list of items.
Input
    <ul>
        <li> Item 1 </li> 
        <li> Item 2 </li> 
        <li> Item 3 </li>
    </ul>
Output
  • Item 1
  • Item 2
  • Item 3


<ol>
<ol> stands for ordered list and is used to make a list that is numbered.
Input
    <ol>
        <li> Item 1 </li> 
        <li> Item 2 </li> 
        <li> Item 3 </li>
    </ol>
Output
  1. Item 1
  2. Item 2
  3. Item 3


<dl>
<dl> for descriptive list, which is used to display data with a pair, a phrase and a description.
Input
    <dl>
    <dt> Item 1 </dt>
            <dd> Item 1 description. </dd>
        <dt> Item 2 </dt> 
            <dd> Item 2 description. </dd>
        <dt> Item 3 </dt>
            <dd> Item 3 description. </dd>
    </dl>
Output
Item 1
Item 1 description.
Item 2
Item 2 description.
Item 3
Item 3 description.

Border-radius
You can add rounded corners to HTML elements using the border-radius property.
Input
    .CSS8 {
    border: 2px solid ;
    border-radius: 20px ;
    text-align: center;
    }
    
    .CSS8_2 {
    border: 2px solid ;
    border-radius: 20px 60px ;
    text-align: center;
    }
    
    .CSS8_3 {
    border: 2px solid ;
    border-radius: 80px 40px 90px ;
    text-align: center;
    }
Output

This is a border-radius = 20 px, which will apply to all sides.

This is a border-radius = 20px and 60px which will be applied on opposite sides respectively.

This is a border-radius = 80px, 40px, and 90px which will apply to all sides, execpt the 2nd value will be applied on two corners- top right, bottom left.
